From your `docker-compose.yml` and `README.md`:

**First time setup:**
```bash
docker compose up -d
```

**Check if it's ready:**
```bash
curl http://localhost:8000/health
```

**Inspect OpenAI backend configuration:**
```bash
curl http://localhost:8000/health/openai
```

**Check Redis cache reachability:**
```bash
curl http://localhost:8000/health/redis
```

**Subsequent starts:**
```bash
docker compose up -d
```

**Stop:**
```bash
docker compose down
```
