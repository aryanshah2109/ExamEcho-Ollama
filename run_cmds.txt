From your `docker-compose.yml` and `README.md`:

**First time setup:**
```bash
docker compose up -d
docker compose exec ollama ollama pull mistral:7b
```

**Check if it's ready:**
```bash
curl http://localhost:8000/health
```

**Subsequent starts:**
```bash
docker compose up -d
```

**Stop:**
```bash
docker compose down
```

The model pull (~4 GB) only needs to be done once since it's stored in the `ollama_data` volume.
